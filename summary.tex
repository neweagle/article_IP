{\em Глазунов~В.\,В., Курочкин~Л.\,М., Курочкин~М.\,А., Попов~С.\,Г.\/} \textbf{Исследование доступности сервисов мобильных пользователей облачно--ориентированой избыточной мультипротокольной сети автомобилей.}

{\small
Совокупность современных автомобилей, дорожной инфраструктуры и облачных информационных ресурсов образуют интеллектуальную транспортную сеть. Целью создания такой сети являются повышение безопасности и комфорта участников дорожного движения.

Обмен сообщениями между автомобилем и облачной средой осуществляется через выделенный канал связи. Посредством этого канала, облачная среда собирает  данные о автомобилях и дорожной обстановке, пользователи получают доступ к службам экстренной помощи, информационным, справочным и развлекательным ресурсам.

Проблемой доступа к сервисам облачной среды является отсутствие непрерывно существующего выделенного беспроводного канала связи, что приводит к отказу в обслуживании. 

Канал передачи данных организуется при помощи автомобильных телематических устройств или персональных устройств пользователя, таких как смартфоны, лэптопы и общедоступные сотовые сети передачи данных.

Решением, позволяющим улучшить качество доступа к сервисам облачной среды, является применение mesh сетей с мультипротокольными сетевыми интерфейсами, обеспечивающими передачу информационных сообщений средствами доступных сетей передачи данных.

Особенностью mesh сетей является динамически изменяемая топология и отсутствие априорной информации о расположении точек доступа к облачной среде, что определяет возможное наличие множества различных путей доставки информационных сообщений.

Основным средством исследования поведения mesh сетей является имитационное моделирование. В рамках данной работы была разработана модель автомобильной mesh сети, реализованной на основе модели движения транспорта IDM средствами дискретно-событийного симулятора компьютерных сетей ns--3.

Анализ результатов моделирования показал что применение автомобильной mesh сети существенно повышает время доступности облачных сервисов обмена экстренными сообщениями в различных конфигурациях автомобильной mesh сети,свойств сетевого трафика и особенностей организации дорожного движения.
}

{\em Glazunov~V.\,V., Kurochkin~L.\,M., Kurochkin~M.\,A., Popov~S.\,G.\/} \textbf{The research service availability of mobile users in cloud oriented multiprotocol network.}

{\small 
A set of modern vehicles, road infrastructure and cloud computing resources forms an intelligent transportation network. The main goal of such network is improving safety and transportation comfort of the traffic participants.
 
Exchange of messages between the vehicle and cloud environment is provided via dedicated network channel. Using this channel cloud environment gathers data about vehicles and road traffic, users gain access to emergency services, information, inquiry and entertainment resources.
 
Problem in access to cloud environment services is absence of persistent dedicated wireless network channel, which leads to denial of service.
 
Data transfer channel can be established with the help of automotive telematic devices or user's personal devices, such as PDA, laptop and easily accessible cellular network equipment.
 
The solution which allows to improve quality of access to the cloud environment services is adaptation of mesh-networks with multiprotocol network interfaces that provide messages exchange using any available data transfer network.

The feature of mesh-networks is dynamically changing topology and absence of a priory information about arrangement of cloud environment access points, all of which defines possible existence of many different routes for information messages delivery.
 
The main method of analysis of the behavior of mesh-networks is simulation which allows to perform different scenarios of mesh-network behavior without implementation of full-scale experiment. Within the bounds of this work the model of automotive mesh-network was developed on basis of IDM traffic model and implemented through NS-3 discrete-event simulator of computer networks.
 
Analysis of simulation results has revealed that use of automotive mesh-network substantially increases availability of emergency messages exchange cloud services in different configurations of automotive mesh network, network traffic properties and road traffic characteristics.
