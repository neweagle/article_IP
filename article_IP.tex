{\footnotesize

\noindent {\it Глазунов~В.\,В., Курочкин~Л.\,М., Курочкин~М.\,А., Попов~С.\,Г.\/} {\bf Исследование доступности сервисов мобильных пользователей облачно--ориентированой избыточной мультипротокольной сети автомобилей.}\\
{\bf Аннотация.} 
Совокупность современных автомобилей, дорожной инфраструктуры и облачных информационных ресурсов образуют интеллектуальную транспортную сеть, сервисы которой, позволяют повысить безопасность и комфорт передвижения участников дорожного движения. Проблемой обеспечения информационного обмена является отсутствие постоянного выделенного беспроводного канала связи. В статье предложено решение, позволяющее увеличить время доступности сервисов облачной среды, состоящее в дополнении индивидуального выделенного канала mesh сетью. В этом случае функции маршрутизации принимает на себя мультипротокольный сетевой узел. Описана имитационная модель сети передачи экстренных сообщений и сформулирован перечень параметров моделирования. Результаты моделирования демонстрируют увеличение времени доступности сервисов облачной среды обмена экстренными сообщениями в различных конфигурациях автомобильной mesh сети, свойств сетевого трафика и особенностей организации дорожного движения.\\
{\bf Ключевые слова:}
автомобильные сервисы, mesh, 802.11s, LTE, IDM, облачные сервисы, мультипротокольный узел, протоколы передачи данных, протоколы маршрутизации, имитационное моделирование, ns--3.

\vskip 2mm

\noindent {\it Glazunov V.\,V., Kurochkin L.\,M., Kurochkin M.\,A., Popov S.\,G.\/} {\bf The research of service availability of mobile users in cloud oriented multiprotocol surplus network of vehicles.}\\
{\bf Abstract.} 
A set of modern vehicles, road infrastructure and cloud computing resources forms an intelligent transportation network, whose services allow to improve safety and transportation comfort of the traffic participants. Absense of persistent dedicated wireless communication channel is a problem for information exchange provision. Solution suggested in the article allows to increase availability of cloud services by addition of mesh network to dedicated channel. In this case routing abilities are provided by the multiprotocol network node. Simulation model of emergency messages transfer network has been built with the set of simulation parameters. The results obtained show increase in availability of emergency messages exchange cloud services in different configurations of automotive mesh network, network traffic properties and road traffic characteristics.\\
{\bf Keywords:}
vehicle services, automotive services, cloud services, cloud environment, multiprotocol node, data transfer protocol, routing protocols, network simulation, vehicle, VIN, mesh, 802.11s, LTE, IDM.
